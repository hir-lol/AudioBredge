# Об проекте
---
**AudioBredge** - это приложение которое передаёт звук с виртуального динамика на другое устройство в локальной сети
---
# Об репозитории 
--- 
В данном репозиториии находиться серверная часть на **python** которая принимает данные с вертуального динамика и отправляет его в сеть. 

В качестиве виртульаного динамика используется [VB-CABLE](https://vb-audio.com/Cable/index.htm)
---
## Работа с исходниками
---
Установка исходников  
``` bash
git clone https://github.com/hir-lol/AudioBredge
```
Установка зависимостей
``` bash
pip install -r requiremnets.txt
```
Документация по общению между сервером и клиентом находиться [здесь](Server/server.md)
